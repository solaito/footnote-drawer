== Usage ==
Use [fnd]sample footnote[/fnd] in your posts and pages.

== Installation ==

1. From the WP admin panel, click “Plugins” -> “Add new”.
1. In the browser input box, type “Footnote Drawer”.
1. Select the “Footnote Drawer” plugin and click “Install”.
1. Activate the plugin.

OR…

1. Download the plugin from this page.
1. Save the .zip file to a location on your computer.
1. Open the WP admin panel, and click “Plugins” -> “Add new”.
1. Click “upload”.. then browse to the .zip file downloaded from this page.
1. Click “Install”.. and then “Activate plugin”.
